ATAECINA PROJECT

Ataecina Project es una suite de herramientas de ciberseguridad diseñada para proporcionar la última línea de defensa en el mundo digital. El proyecto Ataecina se ha creado como una herramienta de código abierto para que en los centros de estudios puedan aprender el funcionamiento de herramientas básicas de ciberseguridad y sentar una base de conocimiento de cómo funcionan tecnologías como SIEM, EDR, WAF, Firewalls, entre otros. Con la suite de Ataecina, los estudiantes pueden testear en un entorno de laboratorio el funcionamiento de un SOC y aprender cómo crear una infraestructura segura a la hora de bastionar una empresa.

La Ciberseguridad se ha vuelto esencial en el nuevo mundo digitalizado en el que vivimos



Esta suite ofrece herramientas avanzadas para la ciberseguridad, facilitando el aprendizaje y el desarrollo de habilidades críticas para la protección digital.



SIEM

Monitoreo centralizado y análisis de eventos de seguridad.

EDR

Protección avanzada contra amenazas en dispositivos finales.

WAF

Defensa para aplicaciones web frente a ataques comunes.

NGFW

Firewall avanzado con inspección profunda de paquetes y más.
